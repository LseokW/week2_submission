Min-Max Normalization

단위가 극단적으로 다른 데이터를 0~1 사이로 압축해
머신러닝 모델이 공정하게 학습할 수 있게 만드는 필수 전처리 기법

⚙ 정규화 원리

Min-Max 공식
$$x' = \dfrac{x - x_{\min}}{x_{\max} - x_{\min}}$$

결과값은 항상 $0 \leq x' \leq 1$ 범위에 들어옵니다.
최솟값 → 0, 최댓값 → 1, 나머지는 비례 배치.

⚠ 정규화 없이 학습하면?
연봉(~1억)이 나이(~60)보다 백만 배 크므로 모델이 연봉에만 집중하고 나이를 무시함
✓ 정규화 후 학습하면?
모든 특성이 0~1 동일 스케일로 맞춰져 모델이 각 특성을 공평하게 반영
📌 numpy 구현 핵심
x' = (x - x.min()) / (x.max() - x.min())

데이터 생성
np.random.seed(42)
salary = np.random.uniform(
    30_000_000, 100_000_000, n)
age = np.random.uniform(20, 60, n)
Min-Max 정규화
# (x - min) / (max - min)
salary_norm = (salary - salary.min()) \
            / (salary.max() - salary.min())
age_norm    = (age - age.min()) \
            / (age.max() - age.min())
정규화 전 통계
# 단위 차이 확인
print(salary.min())   # ~3천만
print(salary.max())   # ~1억
print(age.min())      # ~20
print(age.max())      # ~60
정규화 후 통계
# 모두 0~1 범위
print(salary_norm.min())  # 0.0
print(salary_norm.max())  # 1.0
print(age_norm.min())     # 0.0
print(age_norm.max())     # 1.0
