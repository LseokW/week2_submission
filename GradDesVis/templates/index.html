f(x) = x² 위에서 최솟값을 향해 내려가는 공

● 대기 중

Gradient Descent

산 꼭대기에서 공을 굴리는 것처럼, 기울기의 반대 방향으로
한 걸음씩 이동하며 손실 함수의 최솟값을 찾아갑니다.

f(x) = x² ∇f(x) = 2x 목표: x = 0

⚙ 알고리즘 원리

업데이트 규칙
$$x_{t+1} = x_t - \alpha \cdot \nabla f(x_t)$$
이 예제에서
$$f(x) = x^2 \quad \Rightarrow \quad \nabla f(x) = 2x$$
$$x_{t+1} = x_t - \alpha \cdot 2x_t$$
🐢 lr 너무 작음 (0.01~0.05)
수렴은 하지만 매우 많은 step이 필요. 학습 속도 느림
✓ lr 적당 (0.1~0.4)
빠르고 안정적으로 최솟값에 수렴. 실전에서 권장
💥 lr 너무 큼 (0.8~0.9)
최솟값을 지나쳐 발산. Loss가 무한정 증가
📌 임계값
f(x)=x²에서 lr ≥ 1.0 이면 반드시 발산. lr < 1.0 이면 수렴 보장

{ } 구현 핵심 코드

손실 함수
def loss(x):
    # f(x) = x²
    return x ** 2

def gradient(x):
    # ∇f(x) = 2x
    return 2 * x
업데이트 규칙
# x_new = x - lr·∇f(x)
for step in range(max_steps):
    grad = gradient(x)
    x = x - lr * grad

    if loss(x) < 1e-6:
        break  # 수렴!
발산 감지
# |x| > 50 이면 발산
x_new = x - lr * grad
if abs(x_new) > 50:
    diverged = True
    break  # 💥 발산!
x = x_new
