## 무엇을 배우는 사이트인가?

**비지도 학습(Unsupervised Learning)** — 정답(레이블)이 없는 상태에서 데이터가 스스로 그룹을 찾아가는 과정을 눈으로 확인합니다.

`scikit-learn` 같은 외부 ML 라이브러리를 전혀 사용하지 않고, **numpy만으로 K-Means 알고리즘을 처음부터 구현**했습니다.

---

## 알고리즘 원리

K-Means는 4단계를 반복합니다:

```
1. 초기화  →  데이터에서 K개(=3) 랜덤 중심점 선택
2. 배정    →  각 점을 가장 가까운 중심점에 배정
3. 이동    →  각 클러스터의 평균으로 중심점 갱신
4. 수렴    →  중심점 변화량 < 1e-6 이면 종료
```

### 핵심 수식

**WCSS (Loss 함수)**
$$\text{WCSS} = \sum_{k=1}^{K}\sum_{x \in C_k} \|x - \mu_k\|^2$$
- 각 점과 배정된 중심점 사이 거리의 제곱합
- 값이 작을수록 군집이 촘촘하게 모여있음

**유클리드 거리**
$$d(x, \mu_k) = \sqrt{\sum_{i=1}^{n}(x_i - \mu_{k,i})^2}$$
- 2D 평면에서 두 점 사이의 직선 거리
- 가장 가까운 중심점에 배정할 때 사용

## 사용법

### 1. 군집화 실행
1. 슬라이더로 **최대 반복 횟수** 조절 (1~10회)
2. **군집화 실행** 버튼 클릭
3. 오른쪽에 **군집 결과 그래프** + **Loss 곡선** 표시
4. 왼쪽 뱃지에서 반복 횟수 / 수렴 여부 / 최종 WCSS 확인
5. 중심점 좌표 테이블 (Cluster A/B/C) 확인

### 2. 새 좌표 예측
1. X, Y 입력란에 좌표 입력 (또는 빠른 선택 버튼 사용)
2. **클러스터 예측** 버튼 클릭
3. 예측 결과 그래프에서 금색 별(★)로 위치 확인
4. 배정된 클러스터 이름과 중심점까지 거리 확인
