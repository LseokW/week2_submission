WCSS Loss 함수
$$\text{WCSS} = \sum_{k=1}^{K}\sum_{x \in C_k} \|x - \mu_k\|^2$$

각 점과 배정된 중심점까지 거리의 제곱합. 작을수록 군집이 촘촘함.

유클리드 거리
$$d(x, \mu_k) = \sqrt{\sum_{i=1}^{n}(x_i - \mu_{k,i})^2}$$

2D 평면에서 두 점 사이의 직선 거리. 가장 가까운 중심점에 배정.

배정 단계 (Assign)
# (N,1,2) - (1,K,2) → (N,K)
diffs = data[:, np.newaxis, :] \
      - centroids[np.newaxis, :, :]
dists = np.sqrt((diffs**2).sum(axis=2))
labels = np.argmin(dists, axis=1)
중심점 갱신 (Update)
# 각 클러스터 평균으로 이동
for k in range(K):
    members = data[labels == k]
    if len(members) > 0:
        centroids[k] = members.mean(axis=0)
WCSS 계산
# 클러스터 내 거리 제곱합
wcss = 0
for k in range(K):
    m = data[labels == k]
    diff = m - centroids[k]
    wcss += (diff**2).sum()
수렴 판정
# 중심점 이동량 < threshold
shift = np.max(np.linalg.norm(
    new_centroids - old_centroids,
    axis=1))
if shift < 1e-6:
    converged = True; break
